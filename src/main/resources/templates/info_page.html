Алексей Шамота: разработка API сервиса, принимающего на вход исходный текст с ключом для шифрования. API возвращает в JSON формате зашифрованный текст, который впоследствие отображается на странице

Используемый стек технологий/программ:
Рамазан: Java Spring Boot, JavaScript, Hibernate, POSTMAN, MySQL, Thymeleaf
Алексей: Python Django, ...

Метод шифрования: исключающее или (оно же XOR)
